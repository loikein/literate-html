Accessibility Statement

This is the official accessibility statement for Example Project of Literate HTML.

If you have any questions or comments, feel free to email me at feedback@example.com.

Standards Compliance

All pages on this site are WCAG AA approved, complying with WCAG 2.0 Checklist.
This is a judgement call; many accessibility features can be measured, but many can not.
I have reviewed all the guidelines and believe that all these pages are in compliance.

All pages on this site validate as valid current HTML document.
This is not a judgement call. A program can determine with 100% accuracy whether a page is valid HTML. For example, you can check the home page for HTML validity yourself.

All pages on this site use structured semantic markup.

Visual Design

  • This site uses only relative font sizes, compatible with the user-specified “text size” option in visual browsers.
  • If your browser or browsing device does not support stylesheets at all, the content of each page is still readable.
